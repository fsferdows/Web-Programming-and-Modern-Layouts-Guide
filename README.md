# Web Programming and Modern Layouts Guide
🎓 **A Premium, High-Performance Interactive Study Academy & Playground**

Welcome to the **Web Programming and Modern Layouts Guide**—an advanced, highly polished educational web application designed to master W3C specifications, web request routing, accessibility standards, CSS specificity, and modern multi-dimensional layouts.

This application is built with a deep commitment to high fidelity, adaptive responsiveness, WCAG 2.1 AAA contrast ratios, and tactile interactive playgrounds to transform abstract concepts into tangible mental models.

---

## 🎨 Creative Architecture & Key Features

### 1. Interactive Study Guide (Core Pillars)
Discover and master the 5 structural pillars of web development through interactive step-by-step visualizations and diagnostic panels:
*   **Web Infrastructure:** Interactive DNS Lookup flow simulator mapping out how human-readable URLs resolve into digital IP addresses. Learn the strategic "curb-cut effect" with live broken-image and screen-reader simulators.
*   **CSS Selectors & Syntax:** Build complex selector query arrays and compute exact 3-part specificity vector scores `(ID, Class, Element)` in real-time.
*   **Display Mechanics:** Compare block, inline, and inline-block element flow within a tactile viewport container.
*   **Responsive Design:** Understand viewport meta-tags and fluid rendering rules with live, toggleable mobile device mock renders.
*   **Layout Strategy:** Contrast the spatial dimensions of relative, absolute, fixed, and sticky positioning rules.

### 2. Live Code Playground
A built-in playground that offers a sandboxed rendering preview. Run presets or write your custom HTML and CSS to practice selectors, table cell structures (`colspan`, `rowspan`), or position layouts directly in your browser.

### 3. Visual Layout Lab
A dedicated visual alignment workspace that pits **1D Flexbox Alignment** against **2D CSS Grid Matrix Template** properties. Customize gaps, justify offsets, and instantly export the auto-compiled clean CSS layout code.

### 4. Interactive Practice (Exam & Flashcards)
*   **Revision Flashcards:** Click-to-flip cards designed to review DNS roots, inline baseline mechanics, and container positioning edge cases.
*   **Comprehensive Practice Exam:** A 12-question quiz engine testing core layout theories with detailed explanations and instant accuracy reports.

## 💻 Technical Stack

The application is engineered with a modern, high-performance web stack:
*   **Framework:** [React 19](https://react.dev/) + [Vite 6](https://vite.dev/) + [TypeScript](https://www.typescriptlang.org/) for lightning-fast responsive updates.
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) for fluid responsive units and high-contrast styling.
*   **Animations:** [Motion](https://motion.dev/) for smooth transitions, interactive hover cards, and premium tactile states.
*   **Icons:** [Lucide React](https://lucide.dev/) for precise vector graphics.

## 🛠️ Installation & Local Development

To run the application locally on your machine, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone <your-repository-url>
    cd web-programming-guide
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Start Development Server:**
    ```bash
    npm run dev
    ```
    The application will boot up and be accessible locally.

4.  **Production Build Compilation:**
    ```bash
    npm run build
    ```
